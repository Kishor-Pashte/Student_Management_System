MERN Student Management System

A simple full-stack MERN application to manage students with authentication using JWT.


游댳 Features

    User Registration & Login with JWT authentication
    
    Add students with Name, Roll Number, and Course
    
    View a list of all students
    
    Logout functionality
    
    Clean, responsive UI built with React and inline CSS
    

游눹 Tech Stack

    Frontend: React, Axios, React Router DOM
    
    Backend: Node.js, Express.js
    
    Database: MongoDB (Atlas)
    
    Authentication: JWT
    
    Password Security: bcrypt.js
    

游 Usage

    Visit http://localhost:5173 (or your React port)
    
    Register a new account
    
    Login and start managing students
